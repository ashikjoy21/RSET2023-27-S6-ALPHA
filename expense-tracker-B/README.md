# AI-Powered Smart Expense Tracker & Personal Finance Assistant

An intelligent personal finance management system that automatically tracks expenses and income through SMS parsing, provides AI-powered financial insights, and helps users manage budgets, savings goals, and spending habits.

The system combines automated transaction detection, chatbot-based financial assistance, and detailed analytics to help users make smarter financial decisions.

## Highlights
- AI-powered financial chatbot
- Automatic expense detection via SMS parsing
- Budget monitoring with smart alerts
- Savings goals and wishlist planning
- Calendar-based expense tracking
- Admin dashboard for platform management
## Key Features

### Smart Expense Tracking
- Automatic **expense and income detection through SMS parsing**
- Manual expense and income entry
- Transaction categorization (Food, Travel, Bills, etc.)
- Real-time balance tracking

### Budget & Financial Planning
- Monthly budget limit tracking
- **Notifications when spending exceeds the budget**
- Savings goal management
- Wishlist tracking for planned purchases

### Productivity & Reminders
- **Category-based entry reminders**
- **Calendar view for tracking daily expenses**
- Smart notifications for important financial events

### AI Personal Finance Assistant
- AI chatbot to answer financial queries
- Spending pattern analysis
- Budget recommendations
- Personalized financial insights

### Analytics & Insights
- Expense analytics and reports
- Monthly spending breakdown
- Category-wise spending analysis
- Financial trend visualization

### Admin Dashboard
- View all registered users
- Monitor user expenses and income records
- Manage or remove users
- Platform-wide financial statistics
